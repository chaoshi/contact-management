# contact-management

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

node v6.11.4 and up
npm 3.10.10 and up

## Server

### Installing

```
npm install
```

### Running server

```
node server.js
```

## Client
```
cd client
```
### Installing

```
npm install
```

### Running test

```
npm test
```

### Running dev

```
npm start
```

### Running build

```
npm build
```
